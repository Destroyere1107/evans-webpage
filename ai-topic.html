The Bad

Despite all the good it can do, we as a civilization seem intent on using it for all the wrong reasons. We seem to believe that GenAI is able to completely replace a human being in any context. Spoiler alert: it can't. It's a tool. A very powerful tool, but a tool nonetheless. A tool that can easily hallucinate, write awful code, make weird things.

Any John Q. Public schmuck can use GenAI to make something. Hell, that's what I did with this webpage. But in order to Make something good, an LLM is only as poweful as the human directing it.

If in doubt about using AI, ask yourself: given time and resources, could you do it without AI?

If that answer is no, then don't bother with it. Go learn. Read the documentation. Change until that answer is yes.
