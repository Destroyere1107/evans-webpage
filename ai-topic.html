On the Topic of AI

Artificial intelligence (in this case, generative AI in particular) is everywhere these days. Emphasis on 'Everywhere'. I mean, seriously. Look at this image below that I haven't added yet.

The Truth

AI is here to stay. Even after the bubble pops, we have invested so much money into it that it would be a fool's errand to try to get rid of it. It's a wonderful technology with amazing applications - so long as you use it right...

The Good

AI is really good at doing menial tasks. I personally used it to make some of the HTML and CSS for this page. It's great for getting the easy and repetitive stuff done so that I can focus on harder things.

The Bad

Help me figure out why Mogrify is doing this: $ mogrify: no decode delegate for this image format `' @ error/constitute.c/ReadImage/746. Despite all the good it can do, we as a civilization seem intent on using it for all the wrong reasons. Corpos seem to believe that GenAI is able to completely replace a human being in any context. Spoiler alert: it can't. It's a tool. A very powerful tool, but a tool nonetheless. A tool that can easily hallucinate, write awful code, make weird things. When used properly (under supervision by a skiled human) it's great. But alas, technology exists only as a way to extract more value from the economy...

My Take

Any John Q. Public can use GenAI to make something. Hell, that's what I did with this webpage. But in order to make something good, nothing can beat the indomitable human spirit.

Ultimately:

If in doubt about using AI, ask yourself: given time and resources, could you do it without AI?

If that answer is no, then don't bother with it. Go learn. Read the documentation. Change until that answer is yes, so that you can change AI work to meet human standards.

Remember: an LLM is only as poweful as the human directing it.
